# Contributor Notes

Anyone is welcomed to contribute to this project, as long as the contributions are helpful.

## Translating

You can contribute translations by editing [the translation files in paiagram-ui](./crates/paiagram-ui/assets/locales/).
The translations are [Fluent trasnlation files](https://projectfluent.org), which should be easy to edit and flexible.
Please don't submit any unreviewed machine translations. I absolutely hate them.
